# Pasos Firmes — Sitio Web Principal

Sitio web oficial del proyecto **Pasos Firmes** para visibilizar y acompañar a personas con Charcot-Marie-Tooth (CMT).

## 🌐 URL del sitio

Una vez activado GitHub Pages:
```
https://scarletgalicia-collab.github.io/pasos-firmes-web/
```

## 📁 Estructura

```
pasos-firmes-web/
├── index.html                   ← Página principal (renombrar pasos-firmes-cmt.html)
└── README.md
```

## 🚀 Publicación

1. Sube el archivo `pasos-firmes-cmt.html` a este repositorio
2. Renómbralo a `index.html` (importante para que abra como página principal)
3. Ve a Settings → Pages
4. Source: `main` branch / `(root)`
5. Save y espera 2-3 minutos

## 🔗 Repositorio relacionado

El formulario de inscripción está en un repositorio separado:
- **URL:** `https://scarletgalicia-collab.github.io/pasos-firmes-formulario/`

Los botones "Unirme" de este sitio apuntan directamente a esa URL.
